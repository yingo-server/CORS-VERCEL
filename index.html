CORS 代理服务

通用跨域请求中转网关 · 无服务器架构

请求方法

支持方法 GET POST PUT DELETE OPTIONS 请求头 任意自定义头（透传） 请求体 任意格式（透传）

返回格式

智能识别目标响应类型
  • JSON (application/json) → 直接返回原始 JSON 数据（Content-Type: application/json）
  • 其他文本（HTML, CSS, JS, XML, 纯文本等） → 返回原始文本（保持原 Content-Type）
  • 二进制文件（图片, 音视频, PDF, 压缩包等） → 返回 Data URL（格式：data:<mime>;base64,<base64>，Content-Type: text/plain）

使用示例

获取 GitHub 仓库图片（返回 Data URL）
/api/https://raw.githubusercontent.com/vercel/vercel/main/icon.png
调用 GitHub API（返回 JSON）
/api/https://api.github.com/repos/vercel/vercel
获取 Gitee 仓库文件（返回 Data URL）
/api/https://gitee.com/opengauss/openGauss-server/raw/master/README.md
本服务为通用代理，仅转发请求，不存储任何数据。如有疑问，请查阅项目文档。
